# deeplearning
Website for our research project

## Cloning the repo
1. `git clone https://github.com/madhurakhal/deeplearning.git`

## Setup virtual environment
1. `cd deeplearning`
2. `python3 -m venv .`
3. Activate virutal environment `source bin/activate`
3. `pip install -r requirements.txt`



## Starting webserver
1. `cd webapp`
2. `python app.py`
